Приложение на языке Java - GamepadShop. 
Приложение для получения данных выбраных пользователем в магазине и передача их для составление финального прайса товаров.

Задачи ,которые стояли передо мной во время разработки этого приложения:
1. Создать приложения для магазина игровых контроллеров. 
2. Написать простенькое приложение для укрепение базового знания Java и XML.

Приложение было написано в начале моего обучения. 
Приложение в стадии разработки.
